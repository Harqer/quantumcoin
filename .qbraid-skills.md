# qBraid Skills Reference

## Overview

qBraid is a cloud-based quantum computing platform providing:
- **qBraid Lab** — JupyterLab IDE for quantum development
- **qBraid SDK** — Unified Python SDK for cross-platform quantum circuit transpilation and execution
- **qBraid QIR** — Quantum Intermediate Representation tools (Cirq/QASM3 to QIR)
- **PyQASM** — OpenQASM 3 parser, validator, and unroller
- **qBraid CLI** — Command-line interface for environments, jobs, devices
- **qBraid AI** — AI chat and agent mode for quantum coding assistance
- **REST APIs** — RESTful endpoints for devices and jobs
- **VS Code Extension** — Quantum console and environment manager in VS Code

Documentation Index: https://docs.qbraid.com/llms.txt

---

## Installation & Setup

```bash
pip install qbraid
pip install qbraid-qir    # QIR conversion tools
pip install pyqasm        # OpenQASM 3 tools
```

### Authentication
```python
from qbraid.runtime import QbraidProvider

# Option 1: Direct API key
provider = QbraidProvider(api_key='YOUR_API_KEY')
provider.save_config()  # saves to ~/.qbraid/qbraidrc

# Option 2: Environment variable
# export QBRAID_API_KEY='YOUR_API_KEY'
provider = QbraidProvider()
```

Get API key: https://account.qbraid.com → Account → API Keys → Create API Key

## SDK — Core Modules

### `qbraid.programs`
Extracts and manages metadata from supported quantum program types.

### `qbraid.transpiler`
Bridges different quantum programming IRs through native and customizable circuit conversions.

### `qbraid.passes`
Ensures quantum programs conform to hardware specs through runtime transformations.

### `qbraid.runtime`
Defines abstractions for providers, devices, jobs, and results.

### `qbraid.visualization`
Tools for visualizing quantum circuits and experimental data.

## Quickstart — Transpiler

```python
from qbraid import random_circuit, transpile
from qbraid.interface import circuits_allclose

qiskit_ckt = random_circuit("qiskit")
cirq_ckt = transpile(qiskit_ckt, "cirq")
assert circuits_allclose(qiskit_ckt, cirq_ckt)
```

### ConversionGraph
```python
from qbraid import ConversionGraph, ExperimentType

graph = ConversionGraph()
graph.plot(experiment_type=ExperimentType.GATE_MODEL, legend=True)
```

### Custom Conversions
```python
from qbraid import register_program_type, Conversion

register_program_type(MyType, alias="mytype")
conversion = Conversion("qasm3", "mytype", my_conversion_func)
graph.add_conversion(conversion)
```

## Quickstart — QbraidProvider (Runtime)

```python
from qbraid import QbraidProvider

provider = QbraidProvider()
devices = provider.get_devices()

device = provider.get_device("qbraid:qbraid:sim:qir-sv")
device.metadata()
# {'device_id': 'qbraid:qbraid:sim:qir-sv', 'device_type': 'SIMULATOR',
#  'num_qubits': 64, 'status': 'ONLINE', 'queue_depth': 0}

# Run any supported circuit type
jobs = device.run([qiskit_ckt, braket_ckt, cirq_ckt, qasm3_str], shots=1000)
results = [job.result() for job in jobs]
print(results[0].data.get_counts())
```

### Runtime Options
```python
job = device.run(circuit, shots=100, runtime_options={"key": "value"})
```

## QIR (qbraid-qir)

Converts quantum programs to Quantum Intermediate Representation (QIR / LLVM IR).

```bash
pip install qbraid-qir
```

### Cirq to QIR
```python
import cirq
from qbraid_qir.cirq import cirq_to_qir

q0, q1 = cirq.LineQubit.range(2)
circuit = cirq.Circuit(cirq.H(q0), cirq.CNOT(q0, q1), cirq.measure(q0, q1))
module = cirq_to_qir(circuit, name="my-circuit")
ir = str(module)
```

### OpenQASM 3 to QIR
```python
from qbraid_qir.qasm3 import qasm3_to_qir

program = """
OPENQASM 3;
include "stdgates.inc";
qubit[2] q;
bit[2] c;
h q[0];
cx q[0], q[1];
measure q[0] -> c[0];
measure q[1] -> c[1];
"""
module = qasm3_to_qir(program, name="my-program")
```

### QIR to Squin
```python
from qbraid_qir.squin import load
from pyqir import BasicQisBuilder, SimpleModule

mod = SimpleModule("bell", num_qubits=2, num_results=2)
qis = BasicQisBuilder(mod.builder)
qis.h(mod.qubits[0])
qis.cnot(mod.qubits[0], mod.qubits[1])
squin_kernel = load(mod._module)
squin_kernel.print()
```

### Add QIR to ConversionGraph
```python
from qbraid_qir.cirq import cirq_to_qir
from qbraid.transpiler import Conversion, ConversionGraph

graph = ConversionGraph()
conversion = Conversion("cirq", "qir", cirq_to_qir)
graph.add_conversion(conversion)
```

## PyQASM

OpenQASM 3 parser, semantic validator, and unroller.

```bash
pip install pyqasm
```

### Basic Usage
```python
import pyqasm

# From string
qasm_code = """
OPENQASM 3.0;
include "stdgates.inc";
qubit[2] q;
bit[2] c;
h q[0];
cx q[0], q[1];
c = measure q;
"""
module = pyqasm.loads(qasm_code)

# From file
module = pyqasm.load("example.qasm")

# Validate semantics
module.validate()

# Unroll (expand gates, loops, etc.)
module.unroll()
print(pyqasm.dumps(module))

# Unroll with external gates preserved
module.unroll(external_gates=["custom"])

# Write to file
pyqasm.dump(module, "output.qasm")
```
